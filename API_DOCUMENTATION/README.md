## 🚀 Quick Start for Students

### Step 1: Import to Postman
1. Open Postman
2. Click **Import** → Select `POSTMAN_COLLECTION.json`
3. Click **Import** → Select `POSTMAN_ENVIRONMENT.json`
4. Select the **"ANVNA Care Environment"** from the environment dropdown

### Step 2: Login First
1. Open the `1 - Authentication` folder in the collection
2. Click the **"Login"** request
3. Click **Send**
4. You should see `"success": true`
5. The PHPSESSID cookie is now stored in Postman's Cookie Jar

### Step 3: Test Public Endpoints
- Click **"Get All Medicines"** → Send → You should see a list of medicines
- Click **"Get All Doctors"** → Send → You should see a list of doctors
- Click **"Validate Coupon - SAVE10"** → Send → You should see coupon details

### Step 4: Test Protected Endpoints
1. Get the CSRF token from the browser (see AUTH_FLOW.md for instructions)
2. Set `csrfToken` in the Postman environment
3. Try **"Add Medicine to Cart"**, **"Book Appointment"**, etc.

## 🛡️ Important Notes for Testers

> ⚠️ **Authentication**: This API uses **PHP Sessions** (NOT JWT). You must:
> 1. Login first to get a `PHPSESSID` cookie
> 2. Include that cookie in all subsequent protected requests
> 3. Postman handles this automatically via the Cookie Jar

> ⚠️ **CSRF Token**: All POST/DELETE requests require a valid CSRF token.
> - Get it from the browser meta tag after login
> - Send it via `X-CSRF-Token` header OR in the request body as `csrf_token`

> ⚠️ **Available Coupon Codes**: `SAVE10`, `WELCOME`, `HEALTH20`, `DIAGNO50`, `FLAT200`

> ⚠️ **Rate Limiting on Login**: Max 5 failed attempts per 10 minutes per IP.
